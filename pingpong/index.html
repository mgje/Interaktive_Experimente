Visualisierung des Ping-Pong Spiels

Sie können aus der Navigation (rechts) die Ping-Pong-Simulation für ihr System herunterladen. Vor dem Ausführen dieser Simulation werden Sie von Ihrem Betriebssystem gewarnt, dass es sich um gefährliche Software handeln könnte. In diesem Fall ist die Software nicht gefährlich und kann auf Ihrem System installiert werden. Die Ping-Pong-Simulation wurde mit Processing programmiert. Einen ausführlichen Artikel zum Ping-Pong-Spiel finden Sie unter diesem Link. Den Sourcecode finden Sie auf GitHub. Rückmeldungen und Anregungen sind herzlich willkommen! Martin Guggisberg
