Küzeste Rundreise

Sie können aus der Navigation (rechts) die Rundreise-Simulation für ihr System herunterladen. Vor dem Ausführen dieser Simulation werden Sie von Ihrem Betriebssystem gewarnt, dass es sich um gefährliche Software handeln könnte. In diesem Fall ist die Software nicht gefährlich und kann auf Ihrem System installiert werden. Die Rundreise Simulation wurde mit Processing programmiert. Den Sourcecode finden Sie auf GitHub. Rückmeldungen und Anregungen sind herzlich willkommen! Martin Guggisberg
